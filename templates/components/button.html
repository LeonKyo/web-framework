To use this component, simply add the .btn class to a <a> or <button> element.

Example:
<a> Button
Markup:
  1. <a href="#" class="btn">Button</a>
  2. <button class="btn">Button</button>
  3. <input type="button" class="btn" value="Input Button" />
  4. <input type="submit" class="btn" value="Input Submit" />
Button - Disabled state

Along with the .btn class, add another additional .disabled class to the element.

Example:
<a> Button
Markup:
  1. <a href="#" class="btn disabled">Button</a>
  2. <button class="btn disabled">Button</button>
  3. <input type="button" class="btn disabled" value="Input Button" />
  4. <input type="submit" class="btn disabled" value="Input Submit" />
Button - Full Width state

Along with the .btn class, add another additional .full-width class to the element.

Example:
Full Width <a> Button
Markup:
  1. <a href="#" class="btn full-width">Button</a>
  2. <button class="btn full-width">Button</button>
  3. <input type="button" class="btn full-width" value="Input Button" />
  4. <input type="submit" class="btn full-width" value="Input Submit" />
