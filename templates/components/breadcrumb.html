To use this component, simply add the .breadcrumb class to a <ul> or <ol> element.
Set the current/active page with the .active class.

Example:
  • Home
  • Listing
  • Offers Details
Markup:
  1. <ul class="breadcrumb">
  2. <li><a href="#">Home</a></li>
  3. <li><a href="#">Listing</a></li>
  4. <li class="active"><a href="#">Offers Details</a></li>
  5. </ul>
