Button - Active state

To use this component, simply add the .btn class to a <a>, <button> or <input type="button"> element.

Example:
<a> link disguised as a button
Markup:
  1. <a role="button" aria-disabled="false" href="#" class="btn">link disguised as a button</a>
  2. <button role="button" aria-disabled="false" class="btn">Button</button>
  3. <input role="button" aria-disabled="false" type="button" class="btn" value="Input Button" />
  4. <input role="button" aria-disabled="false" type="submit" class="btn" value="Input Submit" />
Button - Disabled state

Along with the .btn class, add another additional .disabled class to the element.

Example:
<a> link disguised as a button
Markup:
  1. <a role="button" aria-disabled="true" href="#" class="btn disabled">link disguised as a button</a>
  2. <button role="button" aria-disabled="true" class="btn disabled">Button</button>
  3. <input role="button" aria-disabled="true" type="button" class="btn disabled" value="Input Button" />
  4. <input role="button" aria-disabled="true" type="submit" class="btn disabled" value="Input Submit" />
Button - Full Width state

Along with the .btn class, add another additional .full-width class to the element.

Example:
Full Width <a> link disguised as a button
Markup:
  1. <a role="button" aria-disabled="false" href="#" class="btn full-width">Button</a>
  2. <button role="button" aria-disabled="false" class="btn full-width">Button</button>
  3. <input role="button" aria-disabled="false" type="button" class="btn full-width" value="Input Button" />
  4. <input role="button" aria-disabled="false" type="submit" class="btn full-width" value="Input Submit" />
