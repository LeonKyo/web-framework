To use this component, simply add the .pagination class to a <ul> or <ol> element.
Set the current/active page with the .active class.

Example:
Table caption
Table Heading Table Heading Table Heading
Table Footer Table Footer Table Footer
Table Data Table Data Table Data
Table Data Table Data Table Data
Table Data Table Data Table Data


For 100% width, simply add the .full-width class to a <table> element.

For borders all around, simply add the .border-lines class to a <table> element.
