To use this component, simply add the .breadcrumb class to a <ul> or <ol> element.
Set the current/active page with the .active class.

Example:
  • Home
  • Listing
  • Offers Details
Markup:
  1. <nav role="navigation" aria-label="breadcrumb">
  2. <ul class="breadcrumb">
  3. <li><a aria-level="1" href="#">Home</a></li>
  4. <li><a aria-level="2" href="#">Listing</a></li>
  5. <li class="active"><a href="#">Offers Details</a></li>
  6. </ul>
  7. </nav>
