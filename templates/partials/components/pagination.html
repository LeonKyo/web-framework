To use this component, simply add the .pagination class to a <ul> or <ol> element.
Set the current/active page with the .active class.

Example:
  • «
  • 1
  • 2
  • 3
  • 4
  • »
Markup:
  1. <nav role="navigation" aria-label="pagination">
  2. <ul class="pagination">
  3. <li><a href="#" class="pagination-prev">&laquo;</a></li>
  4. <li><a href="#" class="active">1</a></li>
  5. <li><a href="#">2</a></li>
  6. <li><a href="#">3</a></li>
  7. <li><a href="#">4</a></li>
  8. <li><a href="#" class="pagination-next">&raquo;</a></li>
  9. </ul>
  10. </nav>
Pagination - Disabled buttons

Simply add the .disabled class to a <a> element.

Example:
  • «
  • 1
  • 2
  • 3
  • 4
  • »
Markup:
  1. <nav role="navigation" aria-label="pagination">
  2. <ul class="pagination">
  3. <li><a href="#" class="pagination-prev disabled">&laquo;</a></li>
  4. <li><a href="#" class="active">1</a></li>
  5. <li><a href="#">2</a></li>
  6. <li><a href="#" aria-disabled="true" class="disabled">3</a></li>
  7. <li><a href="#" aria-disabled="true" class="disabled">4</a></li>
  8. <li><a href="#" aria-disabled="true" class="pagination-next">&raquo;</a></li>
  9. </ul>
  10. </nav>
