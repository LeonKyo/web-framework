Usage:
<div class="page-wrapper">

Full width content are placed here, such as carousel, hero banners, full-width background colour or image, border lines, etc.
The <section> tag uses a fixed width specified in the global configuration SCSS file.
You can edit the $max-container-width variable found in the _config_global.scss file.
You can add a .full-width CSS class to change the <section> tag to 100% width.

<div class="page-wrapper">
 <section>
  <div class="page-container">

The "page-container" class, comes with a default of 20px padding for the sides. Use helper CSS classes to adjust the top and bottom padding. Refer to the Example below for a list of helper CSS classes for inner layout placement.

Markup:
  1. <div class="page-wrapper">
  2.  <section>
  3.   <div class="page-container"></div>
  4.  </section>
  5. </div>

  1. <div class="page-wrapper">
  2.  <section class="full-width">
  3.   <div class="page-container"></div>
  4.  </section>
  5. </div>
