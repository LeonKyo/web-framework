Usage:
<div class="columns col-2">

To use this component, simply add the .columns class to a <div> element.
Followed by using CSS helper class to define the total number of columns required.
For example, if there's 2 columns, the "col-2" CSS class is placed.

<div class="column">

The inside columns, must be within the div tag of ".column" CSS Class.

Markup:
  1. <div class="columns col-2">
  2.  <div class="column">Column 1</div>
  3.  <div class="column">Column 2</div>
  4. </div>

  1. <div class="columns col-3">
  2.  <div class="column">Column 1</div>
  3.  <div class="column">Column 2</div>
  4.  <div class="column">Column 3</div>
  5. </div>

Markup:
  1. <div class="columns col-2 collapsed">
  2.  <div class="column">Column 1</div>
  3.  <div class="column">Column 2</div>
  4. </div>
