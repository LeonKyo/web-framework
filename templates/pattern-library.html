Pattern HTML Structure

All main components will use the following HTML container structure. Components within components will not need this container HTML.

For full width layout with boxed width section
CSS Styles to be implemented here are:

  • Background Image
  • Background Colour
  • Borders

For full width layout with full width section
CSS Styles to be implemented here are:

  • Background Image
  • Background Colour
  • Borders

For boxed width layout with boxed width section
CSS Styles to be implemented here are:

  • Background Image
  • Background Colour
  • Borders

For boxed width layout with full width section
CSS Styles to be implemented here are:

  • Background Image
  • Background Colour
  • Borders
Example HTML Code:
<div class="page-wrapper">
 <div class="page-container full-width">
   <section>For full width layout with boxed width section</section>
 </div>
 <div class="page-container full-width">
   <section class="full-width">For full width layout with full width section</section>
 </div>
 <div class="page-container">
   <section>For boxed width layout with boxed width section</section>
 </div>
 <div class="page-container">
   <section class="full-width">For boxed width layout with full width section</section>
 </div>
</div>

Breakpoints (For Responsive)

The following table shows an overview of available breakpoints and it's respective devices.

Size Breakpoints Device
Mini up to 479px Phones portrait
Small 480px to 767px Phones landscape
Medium 768px to 959px Tablets portrait
Large 960px to 1199px Desktops & tablets landscape
Xlarge 1200px and larger Large Desktops

Usually we build based on 3 breakpoints only, having Mini, Small and Large (which is 768px and larger).

Components Introduction

The components we've built are based on different categories depending on their purpose & functionality.

Category Descriptions
Default / Base These are the basic base components used across within most components.
Layout Fully responsive fluid designs, consists of mainly common content layouts.
Navigation Navigation bars and menus, breadcrumbs and pagination type of components.
Common Common components such as buttons, icons, etc.
JavaScript Components that are configured and controlled by JavaScripts.
